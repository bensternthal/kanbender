kanbender
=========

Kanbanery Helper For Moving Merged Cards To Done Column


Setup
-----

1. Get the source:

   ```sh
   $ git clone git@github.com:bensternthal/kanbender.git
   $ cd kanbender
   ```

2. Install requirements:

   ```sh
   $ npm install
   ```

3. Configure the app:
Copy local.json-dist to local.json and populate the values.
